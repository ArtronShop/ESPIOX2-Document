  ข้อมูลฮาร์แวร์

  • แนะนำบอร์ด ESPIOX2
  • การเข้าสู่โหมดอัพเดทเฟิร์มแวร์แบบแมนนวล

  Arduino IDE

  • เริ่มต้นพัฒนาเฟิร์มแวร์ EPS8266 ด้วย Arduino IDE
  • การใช้งาน ESP8266 ควบคุม LED ผ่าน MQTT โดยใช้ Arduino IDE (C++)
  • การใช้ ESP8266 ทำเว็บเซิฟเวอร์แบบง่ายๆ

  NodeMCU Firmware

  • พัฒนา ESP8266 ด้วยภาษา Lua

  MicroPython Firmware

  • พัฒนา ESP8266 ด้วยภาษา Python

  Espruino Firmware

  • พัฒนา ESP8266 ด้วยภาษา JavaScript
